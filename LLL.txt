git 아이디, 이메일 설정
>git config --global user.name iyeohan
>git config --global user.email leejisoo338@gmail.com

폴더("git-test") 만들기
>mkdir test

만든 폴더("git-test")로 경로 옮기기
>cd git-test

저장소 만들기
>git init

파일(.txt) 만들기
>notepade

----- 파일 이름 정하고 폴더("git-test")에 저장
LLL.txt ----- 파일 이름

깃에 파일 올리기
>git add .  ----- 파일에 있는 모든 파일.
>git add LLL.txt ----- LLL 이라는 파일만.

커밋하기
>git commit -m "message" ----- "message" 부분은 로그.

첫 push일 경우 
git push -u https://github.com/Basic-Computer-Literacy/git-test-gg.git main

branch(main, master) 만들기 -- 요즘엔 'main'을 사용
>git branch main

//저장소 생성 및 연결
$ git init
$ git remote add origin [원격저장소 주소]
$ git branch -m master main

//파일 업로드
$ git pull (또는 git pull origin [브랜치 이름])
$ git add .
$ git commit -m "commit message"
$ git push (또는 git push origin [브랜치 이름])

//추가적인 명령어
$ git remote -v
$ git remote rm origin
$ git branch
$ git config --global init.defaultBranch [브랜치 이름]
$ git status
$ git rm --cached -r .
$ git push -u origin main
